FAQ Heart Rotation Component - Usage Guide

Basic FAQ Structure:

Your FAQ Question Here

Your FAQ answer content goes here...

Another FAQ Question

Another FAQ answer...

Implementation Instructions:

Step 1: Include Files

Add these lines to your HTML head section:

<link rel="stylesheet" href="faq-heart-rotation.css">
<script src="faq-heart-rotation.js"></script>

Step 2: Required Heart SVG Files

Make sure you have these files in your images folder:

  • /wp-content/themes/aenfinite.com/images/heart.svg - Normal heart (for open FAQs)
  • /wp-content/themes/aenfinite.com/images/heart-rotate.svg - Rotated heart (for closed FAQs)

Step 3: HTML Structure

Use the exact HTML structure shown above. Key points:

  • Main container: .faq-container
  • Each FAQ item: .faq-item (add active class for initially open FAQ)
  • Clickable header: .faq-header
  • FAQ title: .faq-header-title
  • Icon container: .faq-header-icon
  • Heart icon: .faq-heart-icon (img tag)
  • Content area: .faq-content
  • Text content: .faq-content-text

Step 4: Heart Icon Animation Behavior

  • Closed FAQ: Shows heart-rotate.svg rotated 90° to point downward
  • Open FAQ: Shows heart.svg at 0° pointing upward
  • Transition: Smooth 0.4s cubic-bezier animation between states
  • Click behavior: JavaScript automatically switches between images and applies CSS rotation

Step 5: Customization

You can customize the component by modifying the CSS variables:

  • Heart icon size: .faq-heart-icon width/height
  • Animation timing: transition properties
  • Colors and spacing: .faq-header, .faq-content styles
